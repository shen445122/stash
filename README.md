# stash

Stash 自用配置与示例集合。

当前内容：

- `collapsed-tiles.stoverride`：折叠面板示例配置
- `examples/collapsed-tiles.yaml`：对应的原始 YAML 示例
- `tiles/`：如后续需要自托管脚本，可放对应 JS

## 用法

将 `collapsed-tiles.stoverride` 导入 Stash / Stash iOS / Stash Mac 对应配置环境中使用。
